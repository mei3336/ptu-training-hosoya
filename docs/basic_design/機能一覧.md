# 機能一覧

| 機能ID | 機能名 | 概要 | 利用者 |
|----------|----------|----------|----------|
| F01 | ログイン機能 | メールアドレス・パスワードによる認証 | 一般ユーザー・管理者 |
| F02 | ログアウト機能 | セッション破棄によるログアウト | 一般ユーザー・管理者 |
| F03 | パスワード管理機能 | パスワードのハッシュ化保存 | システム |
| F04 | セッション管理機能 | セッション発行・維持・失効管理 | システム |
| F05 | アクセス制御機能 | 未認証・権限外アクセスの制御 | システム |
| F06 | ログイン失敗管理機能 | ログイン失敗回数の記録 | システム |
| F07 | アカウントロック機能 | ログイン5回失敗時にアカウントをロック | システム |
| F08 | ロック解除機能 | ロック後60分経過で自動解除 | システム |
| F09 | マイページ参照機能 | 自身のプロフィール情報表示 | 一般ユーザー・管理者 |
| F10 | マイページ編集機能 | 氏名・メールアドレス・パスワード・ニックネーム・自己紹介・アイコン画像の編集 | 一般ユーザー・管理者 |
| F11 | 自己退会機能 | 管理者自身のアカウント削除 | 管理者 |
| F12 | メンバー一覧表示機能 | カード形式で全メンバーを表示 | 一般ユーザー・管理者 |
| F13 | ユーザー詳細表示機能 | メンバー詳細モーダル表示 | 一般ユーザー・管理者 |
| F14 | 名簿一覧表示機能 | テーブル形式で全ユーザーを一覧表示する管理者向け画面 名前・ニックネーム検索・権限変更・登録・削除・編集ボタンがある。| 管理者 |
| F15 | ユーザー登録機能 | 新規ユーザー登録（登録時は一般ユーザー固定） | 一般ユーザー・管理者 |
| F16 | ユーザー編集機能 | 他ユーザー情報の代理編集 | 管理者 |
| F17 | ユーザー削除機能 | 他ユーザーの削除（名簿削除） | 管理者 |
| F18 | 入力チェック機能 | 必須・文字数・形式・画像チェック | システム |
| F19 | 重複チェック機能 | メールアドレス重複チェック | システム |
| F20 | 完了メッセージ表示機能 | 登録・更新・削除・権限変更完了メッセージ表示 | システム |
| F21 | 名前検索機能 | 氏名・ニックネームによるリアルタイム検索 | 一般ユーザー・管理者 |
| F22 | 権限変更機能 | ユーザー権限のインライン変更 | 管理者 |
| F23 | アイコン画像アップロード機能 | JPG/PNG画像のアップロード・保存 | 一般ユーザー・管理者 |
| F24 | 管理者二要素認証機能 | TOTP認証による管理者向け二要素認証 | 管理者 |
