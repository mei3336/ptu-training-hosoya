# 機能一覧

| 機能ID  | 機能名                                     | 概要                                                                                                                                                                                                                                                                                                                                                                              | 利用者                   |
| :------ | :----------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :----------------------- |
| **F01** | ログイン機能                               | メールアドレスおよびパスワードによるユーザー認証を行う。                                                                                                                                                                                                                                                                                                                          | 一般ユーザー・管理者     |
| **F02** | ログアウト機能                             | セッションを完全に破棄し、ログイン画面へと遷移する。                                                                                                                                                                                                                                                                                                                              | 一般ユーザー・管理者     |
| **F03** | パスワード管理機能                         | 安全性を担保するため、パスワードをハッシュ化してデータベースに保存する。                                                                                                                                                                                                                                                                                                          | システム                 |
| **F04** | セッション管理機能                         | ログイン状態の維持、およびセッション有効期限（最終操作から60分）の管理を行う。                                                                                                                                                                                                                                                                                                    | システム                 |
| **F05** | アクセス制御機能                           | 未認証ユーザーや、権限のない機能・APIへのアクセスを遮断する。                                                                                                                                                                                                                                                                                                                     | システム                 |
| **F06** | ログイン失敗管理機能                       | 連続した認証失敗回数をアカウント単位で記録する。                                                                                                                                                                                                                                                                                                                                  | システム                 |
| **F07** | アカウントロック機能                       | ログイン認証に5回連続で失敗した場合、アカウントをロックする。                                                                                                                                                                                                                                                                                                                     | システム                 |
| **F08** | ロック解除機能                             | アカウントロック後、60分が経過した時点でロックを自動的に解除する。                                                                                                                                                                                                                                                                                                                | システム                 |
| **F09** | ユーザー情報参照機能<br>（マイページ）     | 自身のフルプロフィール情報（氏名、メールアドレス、ニックネーム、自己紹介、アイコン画像）を参照・表示する。                                                                                                                                                                                                                                                                        | 一般ユーザー・管理者     |
| **F10** | ユーザー情報編集機能<br>（マイページ編集） | 自身のプロフィール情報（氏名、メールアドレス、パスワード、ニックネーム、自己紹介、アイコン画像）を編集・更新する。                                                                                                                                                                                                                                                                | 一般ユーザー・管理者     |
| **F11** | ユーザー情報削除機能<br>（退会機能）       | 自身のアカウントを削除（退会）する。**管理者権限を持つユーザーがマイページにアクセスした時のみ「退会する」ボタンを表示し、実行可能とする（一般ユーザー時は退会ボタンを完全に非表示にして権限制御を行う）**。<br>ボタン押下時に「退会（削除）の警告ポップアップ（モーダル）」を起動し、**「いいえ」** または **「削除します」** の選択肢を表示し、後者を選択した場合のみ実行する。 | 管理者（自身の退会のみ） |
| **F12** | メンバー一覧表示機能<br>（カード形式）     | ログイン後の初期トップ画面。全登録ユーザーの情報を3列などのグリッド状に並べたカード形式（ギャラリービュー）で動的に一覧表示する。カードにはアイコン、ニックネーム、自己紹介が表示され、クリックすると「詳細プロフィール（参照用モーダル）」が起動する。                                                                                                                           | 一般ユーザー・管理者     |
| **F13** | ユーザー登録機能<br>（自己登録・代理登録） | ログイン前の新規自己入会（一般ユーザー登録）、または管理者による他ユーザーの代理追加を行う。**登録入力時の「権限」選択項目を廃止し、登録時はシステム側で一律「一般ユーザー」として登録する**。                                                                                                                                                                                    | 一般ユーザー・管理者     |
| **F14** | ユーザー編集機能<br>（代理編集）           | 管理者が「名簿管理（表形式）」から他ユーザーのプロフィール情報を編集・更新する。編集画面から「権限」の変更項目を削除し、**ニックネームおよび自己紹介（bio）を含む、プロフィール全項目の代理編集・フル保存に特化させる**。                                                                                                                                                         | 管理者                   |
| **F15** | ユーザー削除機能<br>（名簿除外）           | 管理者が他ユーザーのアカウント削除（名簿から除外）を実行する。<br>アクション列の「削除」ボタンをクリックすると「削除の警告ポップアップ（モーダル）」が起動し、選択肢として **「いいえ」** または **「削除します」** を配置する。後者を選択した際にのみ物理削除（または論理削除）を実行する。                                                                                      | 管理者                   |
| **F17** | 入力チェック機能                           | 各フォームの入力バリデーション（必須、文字数、メール形式、パスワード複雑性チェック）を行う。<br>**アイコン画像アップロード時のファイル制限（JPEG/PNG形式のみ、1ファイルあたり2MB以内）も自動判別してチェックする。**                                                                                                                                                              | システム                 |
| **F18** | 重複チェック機能                           | ユーザー登録・編集時に、他ユーザーと重複するメールアドレスの登録を防ぐ。                                                                                                                                                                                                                                                                                                          | システム                 |
| **F19** | 完了メッセージ表示機能                     | 登録・更新・削除処理などの完了時に、画面上に適切なサクセスメッセージを表示する。                                                                                                                                                                                                                                                                                                  | システム                 |
| **F20** | 名簿管理機能<br>（表形式表示）             | 管理者が迅速な実務を行うための専用ダッシュボード。全ユーザー情報を一覧のテーブル形式（ID、氏名、メールアドレス、権限、操作ボタン）で表示する。表から詳細画面への遷移は廃止する。                                                                                                                                                                                                  | 管理者                   |
| **F21** | 権限インライン変更機能                     | 「名簿管理画面（表形式）」の「権限」列ボタンをクリックすると、「権限更新の警告ポップアップ（モーダル）」が起動する。<br>「本当に権限を変更しますか？」の旨の警告とともに、選択肢として **「いいえ」** または **「はい」** を表示し、**「はい」** を選択した際にのみ、その場で一般/管理者を即時に切り替えてデータベースを更新し、画面を再描画する。                                | 管理者                   |
| **F22** | 名前検索（フィルター）機能                 | 「メンバー一覧画面（カード形式）」および「名簿管理画面（表形式）」の両方に配置され、氏名・ニックネームから該当するメンバーをリアルタイムに絞り込み（フィルター）表示する。                                                                                                                                                                                                        | 一般ユーザー・管理者     |
| **F23** | アイコン画像<br>アップロード機能           | マイページ画面（F10）やユーザー編集画面（F14）において、ローカルの画像ファイル（PNG/JPEG、2MB以内）を選択・送信し、サーバー上のストレージにアップロード保存してデータベースのパス（`icon_image`カラム）を更新する。未設定時はデフォルト画像パスを割り当てる。                                                                                                                     | 一般ユーザー・管理者     |
| **F24** | 管理者二要素認証機能                       | **【新規】** 管理者ログイン時に、認証アプリ（Google Authenticator等）に表示される現在の6桁のワンタイムパスワードの入力を求め、バックエンド（Rails）で整合性を検証する機能。                                                                                                                                                                                                       | 管理者                   |
