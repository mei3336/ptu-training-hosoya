# 画面一覧

| 画面ID    | 画面名                             | 対象の利用者         | 概要                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| :-------- | :--------------------------------- | :------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **SCR01** | ログイン画面                       | 一般ユーザー・管理者 | メールアドレスとパスワードによるユーザー認証を行う。新規ユーザー登録画面（SCR06）への遷移リンクを配置する。  |
| **SCR02** | メンバー一覧画面<br>（カード形式） | 一般ユーザー・管理者 | **【新規】** ログイン成功後の共通初期トップ画面 。全登録メンバーを3列などのグリッド状のカード形式（ギャラリービュー）で並べた組織図。<br>ニックネーム、自己紹介、アイコン画像を表示し 、カードをクリックすると「ユーザー情報詳細モーダル（SCR03）」を表示する 。<br>名前（氏名・ニックネーム）での**リアルタイム検索（フィルター）機能**を備える |
| **SCR03** | ユーザー情報詳細モーダル           | 一般ユーザー・管理者 | **【新規】** メンバー一覧画面（SCR02）でカードをクリックした際にポップアップ起動する、他人のフルプロフィール（ニックネーム、自己紹介、アイコン画像等）の**参照専用モーダル**。|
| **SCR04** | マイページ画面| 一般ユーザー・管理者 | 自身のプロフィール情報（氏名、メールアドレス、ニックネーム、自己紹介、アイコン画像、パスワード）の確認および編集・更新を行う。<br>**【権限制御】一般ユーザーログイン時は「退会する」ボタンを完全に非表示にする。管理者ログイン時にのみ、画面下部等に赤色の「退会する（アカウント削除）」ボタンを配置する 。**<br>※管理者退会時は「退会（削除）の警告モーダル」が起動し、**「いいえ」 / 「削除します」** を選択させる。|               
| **SCR05** | 名簿管理画面<br>（表形式）         | 管理者専用           | 全ユーザーの情報をテーブル形式（ID、氏名、メールアドレス、権限、操作ボタン）で一覧表示する、実務に特化したダッシュボード 。詳細画面への遷移は廃止する。<br>名前（氏名・ニックネーム）での**リアルタイム検索（フィルター）機能**を備える 。<br>以下のインラインポップアップ警告を実装する。<br>・**権限列バッジクリック時**：権限更新警告モーダル（選択肢：**「いいえ」 / 「はい」**）<br>・**削除ボタンクリック時**：削除警告モーダル（選択肢：**「いいえ」 / 「削除します」**）  |
| **SCR06** | ユーザー登録画面| 一般ユーザー・管理者 | 新規ユーザー登録を行う画面（一般の自己入会、または管理者の代理登録）。**「権限（一般/管理者）」の選択項目は完全に廃止**され、一律「一般ユーザー」としてシステム側で初期登録される 。|
| **SCR07** | ユーザー編集画面<br>（代理編集）   | 管理者専用           | 管理者が名簿管理画面（SCR05）から他ユーザーのプロフィール情報を編集・更新する画面 。編集フォームから「権限」の選択項目を削除する。<br>一般ユーザーのマイページ編集と同様に、**ニックネームおよび自己紹介（bio）の2項目を含むプロフィール全項目をフル編集・保存できる仕様とする**。|
