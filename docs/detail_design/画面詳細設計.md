# 1. 画面設計

## 1.1 SCR01 ログイン画面

### 1.1.1 画面概要

利用者認証を行う画面。
メールアドレスおよびパスワードを入力し、システムへのログインを実施する。

### 1.1.2 表示項目

| 項目      | 内容          |
| ------- | ----------- |
| システム名   | システムタイトル表示  |
| メールアドレス | ログインID入力欄   |
| パスワード   | パスワード入力欄    |
| ログインボタン | 認証実行        |

### 1.1.3 操作項目

| 項目      | 必須 |
| ------- | -- |
| メールアドレス | ○  |
| パスワード   | ○  |

### 1.1.4 ボタン

| ボタン名     | 処理       |
| -------- | -------- |
| ログイン     | 認証処理を実行  |

### 1.1.5 権限制御

* 未認証ユーザーのみ表示可能
* 認証済ユーザーはSCR02へリダイレクト

### 1.1.6 画面遷移

| イベント      | 遷移先   |
| --------- | ----- |
| ログイン成功    | SCR02 |

***

# 1.2 SCR02 メンバー一覧画面

### 1.2.1 画面概要

登録されているメンバーをカード形式で一覧表示する。
メンバー検索および詳細参照を行う。 

### 1.2.2 表示項目

| 項目      | 内容          |
| ------- | ----------- |
| ヘッダー    | メニュー表示      |
| 検索ボックス  | 氏名・ニックネーム検索 |
| メンバーカード | 各ユーザー情報表示   |
| アイコン画像  | プロフィール画像表示  |
| 氏名      | ユーザー氏名      |
| ニックネーム  | ユーザー表示名     |

### 1.2.3 操作項目

| 項目      | 処理       |
| ------- | -------- |
| 検索ボックス  | リアルタイム検索 |
| メンバーカード | 詳細モーダル表示 |

### 1.2.4 権限制御

* 認証済ユーザーのみ閲覧可能
* 一般ユーザー・管理者共通

### 1.2.5 画面遷移

| イベント          | 遷移先   |
| ------------- | ----- |
| カード押下         | SCR03 |
| マイページ押下       | SCR04 |
| 名簿管理押下（管理者のみ） | SCR05 |
| ログアウト         | SCR01 |

***

# 1.5 SCR05 名簿管理画面

### 1.5.1 画面概要

管理者向けユーザー管理画面。
登録済ユーザーの検索、登録、編集、削除、権限変更を行う。 

### 1.5.2 表示項目

| 項目         | 内容          |
| ---------- | ----------- |
| 検索ボックス     | 氏名・ニックネーム検索 |
| ユーザー一覧テーブル | 登録ユーザー一覧    |
| 権限列        | 一般・管理者表示    |
| 編集ボタン      | 編集画面遷移      |
| 削除ボタン      | 削除モーダル起動    |
| 新規登録ボタン    | 登録画面遷移      |

### 1.5.3 操作項目

| 項目      | 処理         |
| ------- | ---------- |
| 検索ボックス  | リアルタイム検索   |
| 権限バッジ   | 権限変更       |
| 編集ボタン   | SCR07へ遷移   |
| 削除ボタン   | 削除確認モーダル起動 |
| 新規登録ボタン | SCR06へ遷移   |

### 1.5.4 権限制御

* 管理者のみアクセス可能
* 一般ユーザーは403エラー

### 1.5.5 画面遷移

| イベント   | 遷移先   |
| ------ | ----- |
| 新規登録   | SCR06 |
| 編集     | SCR07 |
| メンバー一覧 | SCR02 |
| マイページ  | SCR04 |

***

# 1.6 SCR06 ユーザー登録画面

### 1.6.1 画面概要

新規ユーザーを登録する画面。
管理者による代理登録で利用する。 

### 1.6.2 入力項目

| 項目      | 必須 |
| ------- | -- |
| 氏名      | ○  |
| メールアドレス | ○  |
| パスワード   | ○  |
| ニックネーム  | ー |
| 自己紹介    | －  |
| アイコン画像  | －  |

### 1.6.3 ボタン

| ボタン | 処理     |
| --- | ------ |
| 登録  | 登録処理実行 |
| 戻る  | 前画面へ戻る |

### 1.6.4 権限制御

* 管理者：代理登録可能
* 一般ユーザー：アクセス不可

### 1.6.5 画面遷移

| イベント        | 遷移先   |
| ----------- | ----- |
| 登録成功（管理者登録） | SCR05 |
| 戻る          | 遷移元画面 |

***
